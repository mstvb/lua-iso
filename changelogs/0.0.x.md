## Added

- `Input` has added

  > Input Class allows you to bind functions to specific keys and check for key presses in the update loop.

- `Camera` has added

  > Camera Class provides methods for converting between world and screen coordinates, as well as managing the camera's position and scale.

- `Render` has added

  > Render Class provides methods to rendering objects on screen with coordinates.

- `Demos` has added

  > Demos for Performance Tests / Examples
